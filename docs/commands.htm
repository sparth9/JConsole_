The Core Commands are as follows:

 

help    - used to provide help information

exit    - used to exit the console, same as quit

quit    - used to exit the console, same as exit

hist    - used to display the command history.

show    - to open the last ouput in IE

pwd     - used to print the current directory.

set     - allows to change properties that affect behaviour of other commands

          or list properties with the pattern

rmdir   - used to remove a directory.

ls      - used to display the directory listing.

cd      - used to change the console directory.

cp      - used to copy a file.

cls     - used to clear the screen.

ptrace  - used to print the last Error Stact trace

alias   - used to display the command aliases.

home    - cd to the console home dir.
