Overview

 

Jconsole is a powerful Java Console Client written 100% in java. Mainly written for System Administrators , Coders and also written with the intention to be plugged into various other applications.

 

Jconsole is lightweight fully functional with a basic set of very helpful commands. More commands can be coded and the added in minutes. If you CODE some more commands . PLEASE sent it to me so that I can add those commands into the Console.
